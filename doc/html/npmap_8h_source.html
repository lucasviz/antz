94 Terminologies used in Trees[edit]
95 
96 Root - the top most node in a tree.
97 Parent - node that has a child.
98 Siblings - nodes with the same parent.
99 Leaves - nodes with no children.
100 Internal nodes - nodes with at least one child.
101 Degree - number of sub trees of a node.
102 Edge - connection between one node to another.
103 Level - The level of a node is defined by 1 + the number of connections between the node and the root
104 Height - The height of a node is the length of the longest downward path between the node and a leaf
105 Forest - A forest is a set of n >= 0 disjoint trees.
